#69 鍵盤相關事件

2024-01-01 ・ 1 min read DoJS

常見的鍵盤事件可分為「壓下」、「按住」、「放開」三種。

快速了解

常見的鍵盤相關事件有以下三種，在多數情況下會將鍵盤事件註冊在 input 等表單輸入框上。

  • keydown 事件：「壓下」鍵盤按鍵時會觸發 keydown 事件。
  • keypress 事件：除了Shift、Fn、CapsLock 這三種按鍵外，其他按鍵被按住時會觸發，若按著不放則會連續觸發。
  • keyup 事件：「放開」鍵盤按鍵時觸發。
