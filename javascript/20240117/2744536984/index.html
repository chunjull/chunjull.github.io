#85 Ajax：處理方式

2024-01-17 ・ 1 min read DoJS

XMLHttpRequest、Fetch、axios 都是 Ajax 的處理方式。

快速了解

一般來說，Ajax 的處理方式有三種：

  1. XMLHttpRequest。
  2. Fetch。
  3. axios。
    前兩者是原生 JS 寫法，最後的 axios 則是需要額外安裝的 JS 套件。
