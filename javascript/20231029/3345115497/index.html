#5 基本型別：字串

字串型別原本專門儲存、處理文字，樣板字面值的出現，大幅提升字串的靈活度。

快速了解

在 JS 中，文字會以字串表示。

字串使用方式

字串會用一組單引號 ’’ 或雙引號 ”” 包住，且兩者不可混用。

在單引號中包覆單引號，或在雙引號中包覆雙引號都會出現問題。

樣板字面值

樣板字元值（template literal）是 ES6 新增的特殊字串，支援多行字串、允許將變數嵌入字串，且能夠內嵌運算式，大幅提升字串的靈活度。
樣板字元值由反引號 ```` 所包覆，由一般字串、錢字號 $ 、大括弧 {} 組成。
